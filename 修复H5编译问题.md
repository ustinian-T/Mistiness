# 修复 H5 编译错误 - 文件重命名指南

## 问题原因

uni-app x 使用 `.uvue` 和 `.uts` 文件格式，但编译到 H5 需要 `.vue` 和 `.ts` 格式。

## 解决方案：重命名所有文件

### ⚠️ 操作前准备
1. **关闭 HBuilderX 中所有打开的文件**
2. 关闭编译器/运行中的进程
3. 备份项目（或确认已提交到 Git）

### 需要重命名的文件

#### 1. 页面文件（.uvue → .vue）

```
pages/index/index.uvue     → pages/index/index.vue
pages/month/month.uvue     → pages/month/month.vue
pages/keyword/keyword.uvue → pages/keyword/keyword.vue
pages/dynasty/dynasty.uvue → pages/dynasty/dynasty.vue
pages/podcast/podcast.uvue → pages/podcast/podcast.vue
pages/auth/auth.uvue       → pages/auth/auth.vue
pages/profile/profile.uvue → pages/profile/profile.vue
pages/webview/webview.uvue → pages/webview/webview.vue
```

#### 2. 组件文件

```
components/app-footer.uvue → components/app-footer.vue
App.uvue                   → App.vue
```

#### 3. 脚本文件（.uts → .ts）

```
main.uts              → main.ts
utils/flowerData.uts  → utils/flowerData.ts
utils/auth.uts        → utils/auth.ts
```

### 手动操作步骤

#### 方法 1：在 HBuilderX 中重命名
1. 在项目管理器中右键文件
2. 选择"重命名"
3. 修改扩展名

#### 方法 2：在文件管理器中重命名
1. 打开项目文件夹 `H:\刘思冉\Mistiness`
2. 逐个文件修改扩展名
3. Windows 需要先显示文件扩展名：
   - 文件管理器 → 查看 → 勾选"文件扩展名"

#### 方法 3：使用批处理（推荐）

**关闭 HBuilderX 后**，在项目根目录运行：

```powershell
# 重命名所有 .uvue 为 .vue
Get-ChildItem -Recurse -Filter "*.uvue" | Rename-Item -NewName {$_.Name -replace '\.uvue$','.vue'}

# 重命名所有 .uts 为 .ts
Get-ChildItem -Recurse -Filter "*.uts" | Rename-Item -NewName {$_.Name -replace '\.uts$','.ts'}
```

### 修改后需要更新的代码

#### 1. main.ts 中的导入
```typescript
// 修改前
import App from './App.uvue'

// 修改后
import App from './App.vue'
```

#### 2. 所有页面中的 utils 导入
```typescript
// 修改前
import { xxx } from '../../utils/flowerData.uts'
import { xxx } from '../../utils/auth.uts'

// 修改后
import { xxx } from '../../utils/flowerData.ts'
import { xxx } from '../../utils/auth.ts'
```

或者去掉扩展名（推荐）：
```typescript
import { xxx } from '../../utils/flowerData'
import { xxx } from '../../utils/auth'
```

### 完成后

1. 在 HBuilderX 中刷新项目
2. 重新编译：`发行 → 网站-PC Web 或手机 H5`
3. 应该能正常编译了

## 快速批量替换脚本

保存以下内容为 `rename-files.ps1`，**关闭 HBuilderX 后运行**：

```powershell
# 切换到项目目录
cd "H:\刘思冉\Mistiness"

# 重命名所有文件
Get-ChildItem -Recurse -Include "*.uvue","*.uts" | ForEach-Object {
    $newName = $_.Name -replace '\.uvue$','.vue' -replace '\.uts$','.ts'
    Rename-Item $_.FullName -NewName $newName -Force
    Write-Host "已重命名: $($_.FullName) -> $newName"
}

Write-Host "`n✅ 所有文件重命名完成！"
Write-Host "请在 HBuilderX 中刷新项目并重新编译。"
```

运行：
```powershell
powershell -ExecutionPolicy Bypass -File rename-files.ps1
```
