# 上线前安全检查清单

## ✅ 已完成项

### 密码安全
- [x] 管理员密码通过环境变量传入，不在代码中硬编码
- [x] 用户密码使用 PBKDF2 哈希（100,000 次迭代 + 随机 salt）
- [x] 密码哈希永不返回给客户端

### API Key 保护
- [x] 用户 API Key 使用 AES-256-GCM 加密后存储
- [x] 解密只在云函数服务端进行
- [x] 客户端永远拿不到明文 Key

### Token 安全
- [x] JWT 使用 HMAC-SHA256 签名
- [x] Token 30 天有效期，自动过期
- [x] Token 密钥通过环境变量配置

### 数据库安全
- [x] Schema 权限控制：用户只能读写自己的数据
- [x] 管理员操作需 role='admin' 验证
- [x] 邀请码只有管理员可见

### 速率限制
- [x] 每用户每小时最多 20 次 AI 请求
- [x] 防止 API Key 滥用

### 环境变量
- [x] 敏感信息全部通过环境变量配置
- [x] .env.example 提供模板，不含真实值
- [x] 管理员初始密码用完立即删除

- **永不**在公开渠道分享邀请码

## 🎯 推荐配置

### 强密码策略
- 至少 12 位
- 包含大小写字母、数字、特殊符号
- 不使用常见密码或个人信息

### API Key 使用
- 优先使用服务端系统 Key
- 用户自备 Key 需告知额度消耗
- 设置合理的速率限制
